Displays the excerpt from the latest post in a certain category with link to full post

== Description ==

Category Teaser uses a WordPress widget to display the latest post excerpt from a category you choose.  The output can be styled using CSS.

== Installation ==

* Upload the 'category-teaser-widget' plugin folder to your WordPress plugins directory.  
* Activate the plugin via the WordPress plugins menu.  
* Add the widget to your theme using the Widgets page and customize the options.

== Frequently Asked Questions ==

= Aren't there already other plugins that do this? =

Probably, but I couldn't find any quickly and decided to write the plugin myself using the new WordPress 2.8 Widgets API.  It only took me about 30 minutes to write so it was easier for me to just do it myself then search around for the same plugin features.
